# Programs assets

Place your program files here:

## PDFs
Put your 17 program PDFs in `pdfs/`:
- `public/programs/pdfs/program-1.pdf`
- `public/programs/pdfs/program-2.pdf`
- ... etc

Accessed as: `/programs/pdfs/program-1.pdf`

## Images
Put program thumbnail images in `images/` (these will be clickable and link to YouTube):
- `public/programs/images/program-1.jpg` (or .png)
- `public/programs/images/program-2.jpg`
- ... etc

Accessed as: `/programs/images/program-1.jpg`

Once loaded, we can proceed with the Programs feature implementation.
